# Astro Custom Starter

## Prerequisites

- NodeJS >= 18
- pnpm >= 8

## Built-in Features

- Astro
- React
- Typescript
- Tailwind CSS
- Zod
- Eslint + Prettier
- Nano Stores

## How to Start

- Clone the Repo.
- Create `.env` file at root of the project (see file .env.example).
- `pnpm i` to install dependencies.
- `pnpm dev` to start server at `localhost:4321`
- Let's Go!
